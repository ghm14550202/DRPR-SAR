# DRPR-SAR 工作简介

本工作关注合成孔径雷达（SAR）自动目标识别（ATR）中的对抗鲁棒性问题。深度神经网络虽然在 SAR ATR 任务中表现突出，但容易受到对抗扰动影响，导致模型在部署时面临可靠性风险。

现有防御方法通常依赖输入净化或对抗训练，试图直接抑制扰动带来的影响。然而，这类方法往往缺少对扰动在特征表示空间中如何被编码的显式控制，因此在更复杂的自适应攻击下仍可能失效。

DRPR-SAR 从新的角度重新理解这一问题：对抗扰动主要破坏与分类强相关的判别性特征，而冗余信息相对更稳定，并且仍然保留一定的类别预测能力。基于这一观察，本文提出“扰动路由”思想，不再单纯压制扰动，而是主动引导扰动进入可控的特征分支。

方法上，DRPR-SAR 使用任务引导解耦模块将输入图像分解为冗余流和判别流。该模块基于带旋转增强的 VQ-VAE 实现，用于获得更适合 SAR 目标识别的解耦表示。随后，动态扰动路由模块促使攻击引起的变化主要集中到判别流中，同时通过知识蒸馏让冗余流分类器学习干净教师模型中的类别语义，从而提升模型在攻击场景下的稳定性。

实验结果表明，DRPR-SAR 在多个公开 SAR 数据集和多种攻击设置下，相比代表性防御方法取得了更好的鲁棒识别性能。这说明信息解耦与主动扰动路由相结合，可以作为 SAR ATR 对抗防御的一种有效新范式。
